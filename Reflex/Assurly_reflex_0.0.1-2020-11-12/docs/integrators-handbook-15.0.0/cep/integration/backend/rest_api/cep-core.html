/cep-core/assessment/{assessmentId}

GET Authorized Internal

Loads and assesses the risk assessment questionnaire.

Request
Parameters
Name Required Located In Description Default
assessmentId ✔ path The assessment id to load the assessment data from -
lang query The language that should be used to assess the questionnaire (overrides the language from the provided data block). -
Response

Content-Type: application/json

Status Code Schema Description
200 RiskAssessment The assessment data as seen by the user.
404 No assessment for the given assessmentId.
500 Internal Server Error
502 No suitable knowledgebase found

/cep-core/documents/{assessmentId}

GET Authorized Internal

Creates a PDF document using the DCS (and the document template configured there) and returns it as data:application/pdf;base64. This Service can be used to provide the end-customer using CEP questionnaire with a link to download the PDF directly (via cep-pdf-link directive). For other (internal) uses (e.g. PDF Documents generated in back-office, please use the ReFlex DCS (Document Creation Service) directly.

Request
Parameters
Name Required Located In Description Default
assessmentId ✔ path The assessment id. -
encoded query encoded True, if the response should be Base64 encoded -
doctype query The document template to use to create the PDF -
filename query The document name template to use to name the PDF -
Response

Content-Type: application/pdf

Status Code Schema Description
200 byte[] A response containing the pdf document as byte array and the Content-Disposition header set to 'attachment'.
404 Not Found
