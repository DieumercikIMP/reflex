Answer

The data for an answer to a question given in a questionnaire section of a provided data block.

Name Type Required Description
answer Object The actual answer object. The format depends on the type of the question.
overwriteMode string(one of overwrite-own, overwrite-all, overwrite-none) The overwrite mode for this provided answer. Defaults to 'overwrite-none'.

AnswerModificationInfo

Information about the modification of an answer given to a question. It is recorded who modified the answer and when it was modified the last time.

Name Type Required Description
source string The source of the modification, indicating who modified the answer.
timestamp string(date-time) The timestamp at which the answer was modified.

AssessmentFactor

An assessment-factor, representing a risk that needs to be considered during a risk assessment.

Name Type Required Description
id string The globally unique ID of this assessment-factor.
label string The text with which this assessment-factor is to be presented.
inferred boolean A flag, indicating whether this assessment-factors has been automatically inferred by the RAS engine.
addedBySearchVariable boolean A flag, indicating whether this assessment factor has been added by the RAS engine because it was selected in a search question.
category string The ID of the assessment category that this assessment-factor belongs to.

AssessmentFactorSearchResult

A result of an assessment-factor search.

Name Type Required Description
results AssessmentFactor[] A result of an assessment-factor search.

AssessmentInitialization

A model for all data required to initialize an assessment.

Name Type Required Description
language string ✔ The language code that should be used for this assessment. The specified language must be supported by the knowledgebase for the given product.
subject string The subject that should be used to authorize a user token against this assessment. If this field is left empty, the CEP will create a token that can only be used for this assessment and return it in the IntegrationInfo.cookies object.
provided ProvidedData
paginationMode string(one of Questionnaire, QuestionGroup) The paginationMode that should be used to display the assessment questionnaire in the Browser. Defaults to 'Questionnaire'

Question

Model for a Question in a Questionnaire. This type may required further fields that are relevant for the RAS.

Name Type Required Description
id string The globally unique ID of this questionnaire element.
caption string The text to be displayed for this questionnaire element.
isNew boolean The flag is its a new question or not
answer Object The answer of this questionnaire element. The type of this field depends on the questions type.
answerModification AnswerModificationInfo Information about the modification of the answer given to this question.
answerState string(one of absent, missing, valid, invalid, incomplete) The answering state of the question, specifying whether an answer was given - and if it was given whether it is valid or invalid. Possible states and transition between states apply according to the state model shown in the following diagram.
assignedTo string The role to which the question is assigned to, indicating who is intended to answer this question.
type string The type of this questionnaire element.
declarativeStatements string[] A list of declarative statements for this question. Declarative statements are only supported for 'valid' risk assessments. In case of a validation failure (invalid risk assessment) no declarative statements are delivered.
questions Question[] The child elements directly contained in this questionnaire element.
elements Question[] The nested questions that this structure contains.
messages ValidationMessage[] A list of possible validation/conflict messages for this question.
scope string The logical scope that this questionnaire element is defined in.
searchContextQuestions string[] If this Question is a context sensitive assessment factor search question, this field will be filled with the context question ids.
searchId string The ID of this question that must be specified when searching for assessment factors that are to be added as * answers to this question.
meta Meta
unit string The optional unit of this (number) question.
initialAnswerByProductChange boolean Indicating if the answer for the question has been initially set by performing a product change request

Questionnaire

Name Type Required Description

meta Meta

RiskAssessment

Name Type Required Description
questionnaire Questionnaire The questionnaire, containing questions to be asked and answered during a risk assessment, as well as elements used for structuring the questions.
products Product[] The selected products for which a risk assessment is to be performed.
completed boolean A flag, indicating if the risk assessment is completed. A risk assessment is considered to be completed if all required questions have been validly answered and there are no validation errors for products (i.e. the valid flag is true).
meta ApplicationMeta The optional meta data applied to the risk profile.
results AssessmentResults The results of the risk assessment.
provided ProvidedData[] Data provided by external sources that is relevant for a risk assessment and should be automatically processed.
language string The language used for performing the risk assessment. All data in the risk profile that have translations will be translated in this language. The language is a required property that must be specified by clients. The language is a string that must conform to the following pattern.
links Link[] The optional hypermedia links provided by the REST resource that processed / provided the risk profile.
