Logging

The sample configuration for the CEP Core contains a logback-spring.xml which has been configured for production environments and only logs errors and warnings.

Beyond this, the logging aims to follow the ReFlex logging rules regarding the following log levels:

Info
An expected event occurs that is of interest to non-developers, too. E.g. entry and exit timestamps of a service call so an administrator can tweak the application. Additionally components should inform about their life cycle (e.g. starting, running, shutting down, stopped) using INFO level.
Warning
An event occurred indicating irregular circumstances but the application is still able to process the request and can provide a meaningful response or providing a reasonable solution. The cause does not affect system stability, availability or performance. Log messages with this log level might be collected and periodically sent to an administrator to look after the problems.
Error
The application cannot process a single request due to an unexpected event. The application cannot answer the request with a meaningful response, i.e. it responds with an error message. Exceptions that the system cannot handle (e.g. NullPointerException, OutOfMemory) fall into this category. A log message with this log level basically alerts an administrator to look after the problem.

The configuration of Log4j is documented at its project website.

Sample logback-spring.xml

<?xml version="1.0" encoding="UTF-8"?>
<configuration>
	<include resource="org/springframework/boot/logging/logback/base.xml"/>
	<appender name="FILE" class="ch.qos.logback.core.rolling.RollingFileAppender">
		<!--See http://logback.qos.ch/manual/appenders.html#RollingFileAppender-->
		<!--and http://logback.qos.ch/manual/appenders.html#TimeBasedRollingPolicy-->
		<!--for further documentation-->
		<file>${REFLEX_LOG_ROOTDIR}/cep-core.log</file>
		<encoder>
			<pattern>${FILE_LOG_PATTERN}</pattern>
		</encoder>

		<rollingPolicy class="ch.qos.logback.core.rolling.TimeBasedRollingPolicy">
			<fileNamePattern>${REFLEX_LOG_ROOTDIR}/cep-core.log.%d{yyyy-MM-dd}</fileNamePattern>
		</rollingPolicy>
	</appender>

	<logger name="com.hannoverre.reflex.cep" level="INFO"/>
	<logger name="com.hazelcast" level="WARN"/>
	<logger name="com.hibernate" level="WARN"/>
	<logger name="org.springframework" level="WARN"/>
	<root level="ERROR">
		<appender-ref ref="FILE"/>
	</root>
</configuration>
