Export via Dashboard

MI allows you to export all or selected data from MI manually for further analysis in your preferred BI tools like Microsoft Excel or PowerBI. You may store the exported CSV files in the database of your choice or store the files locally on a hard drive for direct import into your BI tools.

The data are exported as 6 separate CSV files.

  • assessmentFactors.csv - contains information about the assessment factors contained in each application
  • coverresults.csv - contains information about the results per cover (and their attributes and decision reasons)
  • headerinformation.csv - contains meta-information about each application such as dates (creation date/upload date/last changed), status (valid/complete)
  • productresults.csv - contains information about the results on the product level
  • products.csv - contains product information for the applications
  • questionAndAnswers.csv - contains the questionnaire data (questions and answers) of each application

Authentication and Systems

The authentication is done by a cedent - users - roles concept for system usage and access management.

Steps by step guide

Step 1: To export these files, click on “Export” in MI.

Step 2: Then select “all data” if you want to export all the data for a cedent stored in MI for export. Otherwise, you may select a saved query from the drop-down menu to export only selected data from a query.

Once you have selected either “all data” or a particular saved query and clicked on “Export”, you should click on “Copy to clipboard”.

“Copy to clipboard” will copy a URL that you should paste into your browser’s address bar. Then a zip file containing the CSV file will be downloaded.

At the moment, you will most likely need to paste the URL into your browser’s address bar multiple times until the zip file containing the application data is downloaded. This may take several minutes if you are downloading very large data sets. For large data sets, you may need to wait a minute or two before pasting the URL in the address bar before you are able to download the zip file.

Step 3: To export only specific queries, you should first filter the data according to your requirements by selecting, for example, the applications that were referred to underwriting this month on the welcome page of MI. After you save the query and give it a name (“Referred This Month”), you may now select this query from the export drop-down.

Step 4: Once you export the CSV files, you may then import them into a database of your choice, or you may save the files locally on your hard drive for use in Excel or PowerBI.
