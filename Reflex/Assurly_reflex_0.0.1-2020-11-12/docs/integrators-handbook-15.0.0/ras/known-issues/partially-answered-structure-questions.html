Partially answered structure questions

The following issues regarding partially answered structure questions currently exist. A structure question is considered to be partially answered if an element of a structure question is not answered. Note that this also applies to cases where a structure question in nested within a list question.

[PAS-01] Missing error message for invalid assessment

In case a structure question has been answered partially and a risk assessment is then performed, the assessment will become invalid, but there is no error message attached to the question, indicating that an element of the structure question has not been answered.

See also:

  • Usage scenario for invalid assessments involving partially answered structure questions

Workaround: In case of an invalid assessment, clients should also check for incompletely answered questions.

[PAS-02] Missing reflexive questions

In case a structure question has been answered partially and a risk assessment is then performed, there are situations where not all reflexive questions are delivered.

See also:

  • Usage scenario for invalid assessments involving partially answered structure questions

Workaround: Currently there is no workaround available.
