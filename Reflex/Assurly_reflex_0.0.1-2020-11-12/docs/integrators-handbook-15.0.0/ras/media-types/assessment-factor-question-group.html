Assessment factor question group

A questionnaire element that groups a set of questions that belong to an assessment factor of an assessment factor group. This grouping occurs as a result of an assessment factor being added to the risk profile, thereby triggering further reflexive questions introduced by the assessment factor group in which the added assessment factor is contained. The questions for the assessment factor group are provided as child elements within the questions property.

The type specifier for a assessment factor question group is assessment-factor-group. For information about type specifiers see description of media type questionnaire element.

Example (simplified JSON)

{
    "type": "assessment-factor-group",
    "id": "TreatedInHospitalDisease:Diabetes",
    "scope": "ASSESSMENTFACTOR:Diabetes",
    "isNew": true,
    "caption": "diabetes",
    "meta": {...},
    "tags": {...},
    "questions": [{
        "type": "date",
        "id": "Diabetes.DateOfFirstDiagnosis",
        "index": "1.1",
        ...
    },
    {
        "type": "singleselection",
        "id": "Diabetes.HadHospitalStayLast3Years",
        "index": "1.2",
        ...
    },
    {
        "type": "multiselection",
        "id": "Diabetes.TreatmentDiabetes",
        "index": "1.3",
        ...
    },
    {
        "type": "multiselection",
        "id": "SuffersFromFollowingDiseases",
        "index": "1.4",
        ...
    }],
    
    "assessmentFactor": {
        "id": "Diabetes",
        "label": "diabetes"
    }
}

Properties

Note: Described here are only those properties, that are directly related to an assessment factor question group. For a description of general questionnaire element properties, see the description of media type questionnaire element.

Note: An assessment factor question group never has an index.

assessmentFactor (Object)
The assessment-factor that the questions contained in the group belong to.
This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

  • id (String)
    The globally unique ID of the assessment factor.
    This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

  • label (String)
    The text to be displayed for the assessment factor.
    This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

DSL mapping

The assessment factor question group shown in the example above maps to the following (simplified) definition of an assessment factor group in ReFlex/DSL (assuming that the assessment factor Diabetes has been added to the risk profile).

AssessmentFactorGroup DiabetesGroup covers LifeTerm {
    assessmentFactors Diabetes
    
    rules for LifeTerm {
        flow {
            require DateOfFirstDiagnosis
            require HadHospitalStayLast3Years
            require TreatmentDiabetes
            require SuffersFromFollowingDiseases
        }
        
        ...
    }

}
