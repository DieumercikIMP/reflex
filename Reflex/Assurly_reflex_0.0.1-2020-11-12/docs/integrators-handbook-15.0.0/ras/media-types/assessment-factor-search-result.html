Assessment factor search result

The result of a successful search for assessment factors. This is a paged result.

Example (JSON)

{
    "meta": {
        "ras": {
            "knowledgebase": {
                "id": "...",
                "cedent": "...",
                "version": "..."
            }
        }
    },
    "results": [
        {
            "id": "UterusTumour",
            "label": "uterus tumour",
            "category": "Medical"
        },
        {
            "id": "UterusInflammation",
            "label": "uterus inflammation",
            "category": "Medical"
        },
        ...
    ],
    "page": {
        "number": 2,
        "size": 100,
        "totalPages": 3,
        "totalResults": 201
    },
    "links": [
        {
            "href": "/rs/knowledgebases/myknowledgebase/versions/1-0-0/assessmentfactors",
            "rel": ["self"],
            "method": "GET",
            "params": {"questionId": "Diseases", "q": "uterus", "language": "en", "page": "2", "size": "100"}
        },
        {
            "href": "/rs/knowledgebases/myknowledgebase/versions/1-0-0/assessmentfactors",
            "rel": ["prev"],
            "method": "GET",
            "params": {"questionId": "Diseases", "q": "uterus", "language": "en", "page": "1", "size": "100"}
        },
        {
            "href": "/rs/knowledgebases/myknowledgebase/versions/1-0-0/assessmentfactors",
            "rel": ["next"],
            "method": "GET",
            "params": {"questionId": "Diseases", "q": "uterus", "language": "en", "page": "3", "size": "100"}
        }
    ]
}

Properties

meta (ObjectMetaData)

A meta data object containing the information about which knowledge base has been used for the search. Therefore, the ras namespace of the meta data contains a knowledgeBase object consisting of the following properties:

  • id (String) The ID of the knowledge base.
  • version (String) The version of the knowledge base.
  • cedent (String) The cedent’s ID of the knowledge base.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

results (ArrayAssessmentFactor)

The assessment factors that matched the parameters used for the search.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present. If no matching assessment factors have been found by the search, then the array will be empty.

page (ObjectPage)

The page that the matching assessment factors are contained in.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

links (ArrayLink)

The hypermedia links provided by the REST resource that performed the assessment factor search. Links for the following relation types are provided:

  • self
  • next (only if the delivered page is not the first page)
  • prev (only if the delivered page is not the last page)

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.
