Processing info

Meta data information object containing information about the processing of a data element. A processing info object is always contained in the “ras” namespace of a meta data element, having a key named “processing” (see examples below). The primary use for processing info objects is to disclose warning and error conditions that occurred when processing data elements, in order to help integrators to quickly identify the reason of processing errors.

Processing objects may be present on the data elements:

  • Answer data
  • Provided assessment factor

Examples

General structure (simplified JSON)

{
    "error": "...",
    "warning": "..."
}

Processing info contained in a meta data element (JSON)

"meta": {
    "ras": {
        "processing": {
            "error": "Question could not be answered: No assessment factors could be identified to answer the question.",
            "warning": "Assessment factor with ID 'xxx' could not be found."
        }
    }
}

Properties

warning (String)

A message indicating a warning condition that occurred when processing the data element that the processing information is attached to. If more than one warning condition occurred, this property contains the concatenated messages of all warnings.

error (String)

A message indicating an error condition that occurred when processing the data element that the processing information is attached to.
