Invalid risk assessment

Before a risk assessment is processed, it is validated for completeness and correctness. All errors found during this validation are provided as a single problem-json object, aggregated in one top level problem-json of type urn:reflex:ras:errors:invalid-assessment.

The following sections describe possible types of nested problem json objects.

Undefined question
Type urn:reflex:ras:errors:invalid-assessment/undefined-question
Title Question not found

Is returned if the assessment contains a question id that is unknown to the knowledge base against which the current request is made.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/undefined-question",
        "title":"Question not found",
        "detail": "Variable 'IsSmoker' does not exist in KnowledgeBase"
    }]
}

Undefined assessment factor
Type urn:reflex:ras:errors:invalid-assessment/undefined-assessment-factor
Title Assessment factor not found

Is returned if the assessment contains an assessment factor id that is unknown to the knowledge base against which the current request is made.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/undefined-assessment-factor",
        "title":"Assessment factor not found",
        "detail": "AssessmentFactor with ID 'ChainSmoker' does not exist in KnowledgeBase"
    }]
}

Undefined insurance product
Type urn:reflex:ras:errors:invalid-assessment/undefined-insurance-product
Title Insurance product not found

Is returned if the assessment contains a selected insurance product id that is unknown to the knowledge base against which the current request is made.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/undefined-insurance-product",
        "title":"Insurance product not found",
        "detail": "InsuranceProduct with ID 'LifeProduct' does not exist in KnowledgeBase"
    }]
}

Invalid product/rider combination due to language
Type urn:reflex:ras:errors:invalid-assessment/invalid-product-rider-combination
Title The combination of product and rider is not supported due to a missing common language

Is returned if the assessment contains a selected insurance product with a rider attached that do not both support at least one common language in the knowledge base the request is made against.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/invalid-product-rider-combination",
        "title":"The combination of product and rider is not supported due to a missing common language",
        "detail": "The given combination of InsuranceProducts and Riders is invalid, as there is no common available language. Make sure that all Products, mandatory and optional Riders in the request do at least have one common available language."
    }]
}

Invalid question type
Type urn:reflex:ras:errors:invalid-assessment/invalid-question-type
Title The questions type differs from its definition

Is returned if the assessment specifies a question with a different type than the knowledge base the request is made against.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/invalid-question-type",
        "title":"The questions type differs from its definition",
        "detail": "Variable 'Age' was passed by client as type 'Text' but is defined as type 'Number' in the KnowledgeBase"
    }]
}

Invalid question scope
Type urn:reflex:ras:errors:invalid-assessment/invalid-question-scope
Title The questions scope is not valid

Is returned if the assessment contains a question provided with an explicit scope, that differs from the expected scope defined in the knowledge base that this request is made against.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/invalid-question-scope",
        "title":"The questions scope is not valid",
        "detail": "InsuranceProduct with ID 'LifeProduct' used as scope for AssessmentVariable 'IsSmoker' does not exist in KnowledgeBase"
    }]
}

Undefined question type
Type urn:reflex:ras:errors:invalid-assessment/undefined-question-type
Title The question type is unknown

Is returned if the assessment contains a question that specifies a type that is not defined in the knowledge base that this request is made against.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/undefined-question-type",
        "title":"The question type is unknown",
        "detail": "Variable 'IsSmoker' is of unsupported type 'SmokerTypo'"
    }]
}

Calculated question answered
Type urn:reflex:ras:errors:invalid-assessment/calculated-variable-answered
Title The variable is calculated and must not be answered

Is returned if the assessment contains an answer to a question that that is marked as calculated in the knowledge base that this request is made against.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/calculated-variable-answered",
        "title":"The variable is calculated and must not be answered",
        "detail": "Variable 'Age' is defined as calculated Variable in knowledgebase but was provided by the client"
    }]
}

Product selected multiple times
Type urn:reflex:ras:errors:invalid-assessment/product-selected-multiple-times
Title A product has been selected multiple times

Is returned if the assessment contains a selected product more than once.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/product-selected-multiple-times",
        "title":"A product has been selected multiple times",
        "detail": "Duplicate product Life in request"
    }]
}

Duplicate rider selected
Type urn:reflex:ras:errors:invalid-assessment/duplicate-rider-selected
Title A rider has been selected multiple times for the same product

Is returned if the assessment contains a question that has the same rider added more than once.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/duplicate-rider-selected",
        "title":"A rider has been selected multiple times for the same product",
        "detail": "Duplicate rider WaiverOfPremium on product Life in request"
    }]
}

Invalid language
Type urn:reflex:ras:errors:invalid-assessment/invalid-language
Title The selected language is not supported

Is returned if the assessment specifies a language that is not supported by all selected products and riders.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/invalid-language",
        "title":"The selected language is not supported",
        "detail": "Language 'ar' cannot be assessed as it is not a valid available language. Possible available languages for the given InsuranceProduct, mandatory and optional Rider combination are: en, de, fr"
    }]
}

Invalid unit
Type urn:reflex:ras:errors:invalid-assessment/invalid-unit
Title The unit of an answer is not supported by the question

Is returned if the assessment contains an answer with a unit that is not supported by the question the answer belongs to.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/invalid-unit",
        "title":"The unit of an answer is not supported by the question",
        "detail": "'years' is not a valid unit for variable 'weight'. Valid units are: kg, lb"
    }]
}

Invalid selection option
Type urn:reflex:ras:errors:invalid-assessment/invalid-selection-option
Title The given answer is not a valid selection option

Is returned if the assessment contains an answer with a unit that is not supported by the question the answer belongs to.

JSON example

{
    "type":"urn:reflex:ras:errors:invalid-assessment",
    "title":"Invalid risk assessment",
    "detail": "The provided risk assessment is invalid.",
    "nestedProblems": [{
        "type":"urn:reflex:ras:errors:invalid-assessment/invalid-selection-option",
        "title":"The given answer is not a valid selection option",
        "detail": "'Smoker' is not a valid selection option for variable 'gender'. Valid options are: 'Male, Female, Diverse'"
    }]
}
