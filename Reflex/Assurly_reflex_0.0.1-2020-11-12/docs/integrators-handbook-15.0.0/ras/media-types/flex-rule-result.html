FlexRule result

A FlexRule result is an assessment result that contains the execution information for a single FlexRule. Each FlexRule result is contained in the assessment result of a specific insurance product, insurance rider, or insurance cover and contains information about all ratings of the product, rider, or cover that have been available before as well as after the FlexRule has been executed. Additionally, it contains information about all ratings that have been removed from or added to this product, rider, or cover during the execution of the FlexRule.

Example (simplified JSON)

{
    "flexRuleId": "OverallPrioritiesRule",
    "ratingsRemoved": {...},
    "ratingsAdded": {...},
    "ratingsBefore": {...},
    "ratingsAfter": {...},
    "aggregatedRatingBefore": {...},
    "aggregatedRatingAfter": {...},
    "triggered": true,
    "influencedRatings": true,
    "source": "LifeInsuranceProduct"
}

Properties

flexRuleId (String)
The globally unique ID of the FlexRule associated with this FlexRule result.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

ratingsRemoved (ObjectRatings)
The list of ratings that were removed during the execution of the FlexRule associated with this FlexRule result.

This property is set by ReFlex/REST only and must not be modified by clients.

ratingsAdded (ObjectRatings)
The list of ratings that were added during the execution of the FlexRule associated with this FlexRule result.

This property is set by ReFlex/REST only and must not be modified by clients.

ratingsBefore (ObjectRatings)
The list of ratings that were available before the execution of the FlexRule associated with this FlexRule result.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

ratingsAfter (ObjectRatings)
The list of ratings that were available after the execution of the FlexRule associated with this FlexRule result.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

aggregatedRatingBefore (ObjectRating)
The aggregated rating before the flex rule was triggered.

This property is set by ReFlex/REST only and must not be modified by clients.

aggregatedRatingAfter (ObjectRating)
The aggregated rating after the flex rule was triggered.

This property is set by ReFlex/REST only and must not be modified by clients.

triggered (Boolean)
A flag, indicating if the FlexRule associated with this FlexRule result has been triggered during the risk assessment.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

influencedRatings (Boolean)
A flag, indicating if the FlexRule associated with this FlexRule result has influenced any ratings of the insurance product, rider, or cover for which it has been executed. This also includes aggregated ratings.

This property is set by ReFlex/REST only and must not be modified by clients. Clients can expect this property to be always present.

source (String)
Indicates that this flex rule does not actually belong to the embedding entity, but has changed it. For example, if a flex rule belongs to a product and sets an aggregated result on one of its riders, this property will contain the products name. If the flex rule belongs to the embedding entity, it will be missing.

This property is set by ReFlex/REST only and must not be modified by clients.
