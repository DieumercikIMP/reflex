Assessment factors resource

/rs/knowledgebases/{id}/versions/{version}/assessmentfactors

Path Parameters

id The (URL-encoded) ID of a knowledge base.
version The encoded version of a knowledge base. An encoded version must conform to the following pattern: [0-9]+-[0-9]+-[0-9]+(-(SNAPSHOT-[0-9]+-[0-9]+)|RELEASE)?

GET

Searches for assessment factors defined in a knowledge base that match the supplied search parameters. This endpoint uses a paging mechanism.

Query Parameters

name required description
q true The search term that the translations of assessment factors must match. If the search term is blank, an empty result will be returned.
language true The language that the search term is specified in, format according to ISO 639-1 (lower case). If the language is not supported by the knowledge base, an empty result will be returned.
questionId false* The un-scoped ID of the assessment factor search question that is used to match the assessment factors. If no question ID is specified or if it is blank, then this parameter is not taken into account for the search. If the question ID specifies a question that does not exist in the knowledge base or that is not an assessment factor search question, then an empty result will be returned.
category false* The ID of the category that the assessment factors must belong to. If no category is specified or if it is blank, then this parameter is not taken into account for the search. If the category does not exist in the knowledge base, then an empty result will be returned.
questionnaire false The questionnaire containing answered questions that are required for the search. If supplied, this must be a valid JSON representation of a questionnaire. It is recommended to use the minimized information set of a questionnaire. Whether - and which - answered questions are required for the search can be determined by the list of search context questions of the assessment factor search question specified by the supplied question ID. If no question ID is supplied, then a possibly supplied search context questionnaire is ignored. If the search question specified by the supplied question ID requires answered context questions and either no context questionnaire is supplied, or the questionnaire is missing answers for required context questions, then an empty result will be returned.
page false The number of the result page to be retrieved. See pagination query parameters
size false The size of the page, specifying the maximum number of assessment factors to be delivered on a page. The default size is 100, the maximum size is 1000. This parameter must not be specified together with the limit parameter. See pagination query parameters
limit Deprecated: use size instead false The maximum number of matching assessment factors to be delivered. If no limit is specified, a default limit of 100 results will be applied. The maximum limit is 1000. If a larger value is specified, the maximum limit will be used. This parameter must not be specified together with the size parameter.

All query parameters must be properly URL-encoded using the UTF-8 charset.

* Either a questionId or a category must be supplied. Note, that a blank question ID or category is interpreted as having not been supplied.

Responses

Code Name Description Content-Type Media Type Header
200 OK The request was successfully processed. application/json; charset=utf-8 ObjectAssessmentFactorSearchResult
303 See Other The requested page is beyond the last available page n/a Location: URL (without host and context-root) pointing to the last available page.
400 Bad Request The data (i.e. one of the query parameters) passed to this resource is invalid. text/plain; charset=utf-8
414 / 400 / 505 URI Too Long / Bad Request / HTTP Version not supported The URI provided was too long for the web server to process. Typically, this is because of the questionnaire query parameter contains too much data. Note, that some web servers will erroneously respond with code 400 or 505 instead of 414. text/plain; charset=utf-8
404 Not Found The knowledge base with the specified ID and version does not exist. text/plain; charset=utf-8
