Products resource

/rs/knowledgebases/{id}/versions/{version}/products

Path Parameters

id The (URL-encoded) ID of a knowledge base.
version The encoded version of a knowledge base. An encoded version must conform to the following pattern: [0-9]+-[0-9]+-[0-9]+(-(SNAPSHOT-[0-9]+-[0-9]+)|RELEASE)?

GET

Retrieves all insurance products that are defined in a knowledge base.

This subresource provides the products that are available in the specified knowledge base versions (see usage scenario “List products”). The products can be added to a risk assessment.

Query Parameters

name required description
language true The language (ISO 639-1, lower case) in which the returned products are translated. If the translation cannot be found, the default language is used to translate the product caption.

All query parameters must be properly URL-encoded using the UTF-8 charset.

Responses

Code Name Description Content-Type Media Type
200 OK The request was successfully processed. application/json; charset=utf-8 ObjectProductList
400 Bad Request The language parameter in the query passed to this resource is invalid or not set. text/plain; charset=utf-8
404 Not Found The knowledge base with the specified ID and version does not exist. text/plain; charset=utf-8
