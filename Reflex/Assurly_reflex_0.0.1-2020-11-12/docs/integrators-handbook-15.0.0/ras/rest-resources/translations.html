Translation resource

/rs/knowledgebases/{id}/versions/{version}/translations

Provides translations of risk assessments in different languages.

Path Parameters

id The (URL-encoded) ID of a knowledge base.
version The encoded version of a knowledge base. An encoded version must conform to the following pattern: [0-9]+-[0-9]+-[0-9]+(-(SNAPSHOT-[0-9]+-[0-9]+)|RELEASE)?

Query Parameters

name required description
language true The desired language into which you want to translate the risk assessment. A language must conform to the following pattern: ([a-z]{2})(-([A-Z]{2})(-([\w]+))?)? and all products and riders selected in the application must support this language.

POST

Translates a given risk assessment into the language given by language.


Warning

While translating a risk assessment into another language an assessment is performed. Therefore the result of the translated risk assessment may differ from the result of the original risk assessment and additional questions could be asked in cases where additional questions are answered, answers are changed or provided data is given. See known issue TRA-1.


Use this sub-resource with HTTP POST to receive a risk assessment translated into the language given by language. It is a sub-resource of a specific knowledge base version: That knowledge base version is used to process the requests.

Request Body

Content-Type Media Type
application/json; charset=utf-8 ObjectRiskAssessment

Responses

Code Name Description Content-Type Media Type
200 OK The request was successfully processed. application/json; charset=utf-8 ObjectRiskAssessment
400 Bad Request The data passed to this resource is not valid, e.g. because the questionnaire contains a question that does not exists in the knowledge base. text/plain; charset=utf-8
404 Not Found The knowledge base with the specified ID and version does not exist. text/plain; charset=utf-8

Usage

See usage scenario.

Known Issues

  • Risk assessment translation performs an assessment
  • Provided data block is not translated
  • A translation may be associated with information loss
