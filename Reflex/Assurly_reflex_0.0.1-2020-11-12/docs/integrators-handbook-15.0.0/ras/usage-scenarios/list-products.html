List products

A list of all available products can be created with a request to the products resource. The only mandatory parameter is the language.

GET /rs/knowledgebases/myKBname/versions/X-YY-Z-release-20YYMMDD-HHMM/products?language=en

The response contains at its body a list of all available products at "products": [{..}, {..}, ..] with their associated "<product id>".

Example: List of all products for an arbitrary knowledge base.

{
   "meta": {..},
   "language": "en",
   "products": [
      {
         "id": "LifeProduct",
         "caption": "Life Product",
         ..,
         "riders": [
            {
               "id": "TPDownRider",
               "caption": "TPD own",
               "mandatory": true
            },
            {
               "id": "TPDanyRider",
               "caption": "TPD any",
               "mandatory": false
            },
            {
               "id": "ADBRider",
               "caption": "ADB",
               "mandatory": false
            }
         ],
         "typeOfRiskCovered": "LifeTerm",
         "defaultLanguage": "en",
         "languages": ["en", "de"]
      },
      {
         "id": "TPDownProduct",
         "caption": "TPD own",
         "typeOfRiskCovered": "TPDOwn"
         "defaultLanguage": "en",
         "languages": ["en", "de"]
      },
      {..},
      ..
   ]
}

The products of this list are Life Product and TPD own. TPD own is product and also a rider of the product Life Product.

TPD any and ADB are only riders of the product Life Product.
