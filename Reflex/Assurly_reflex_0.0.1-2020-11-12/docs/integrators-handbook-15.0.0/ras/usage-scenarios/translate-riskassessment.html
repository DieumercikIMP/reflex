Translate risk assessment

A risk assessment obtained by a request to the risk assessments resource can be translated into another language by posting it to the translations resource. The URL of the translations resource can be obtained from the link with relation translations in the risk assessment. Its only mandatory parameter is the language. The response contains as its body the risk assessment which has been translated into the requested language.


Warning

While translating a risk assessment into another language an assessment is performed. Therefore the result of the translated risk assessment may differ from the result of the original risk assessment and additional questions could be asked in cases where additional questions are answered, answers are changed or provided data is given. See known issue TRA-1.


Example: Translate a risk assessment.

Assuming the risk assessment in english (see $.language)

                "answer": [

                        "id": "FactorA",
                        "label": "Factor A",
                        "category": "CategoryA"
                    },
                    {
                        "id": "FactorB",
                        "label": "Factor B (Syn)",
                        "category": "CategoryA"
                    }
                ],
                "constraints": {...},
                "meta": {...},
                "index": "4",
                "isNew": false,
                "hasReflexive": true,
                "answerModification": {...},

                "searchLanguage": "en",
                "exposesNextUnusedFactor": false,

            },
            ...

        "hasNewData": false
    },
    "meta": {...},
    "links": [
        ...,
        {
            "href": "/rs/knowledgebases/{knowledgebase}/versions/{version}/translations",
            "rel": [
                "translations"
            ],
            "method": "POST",
            "params": {
                "language": null
            }
        },
        ...
    ],
    "assessmentFactors": [
        ...,
        {
            "id": "FactorA",
            "label": "Factor A",
            "category": "CategoryA"
        },
        {
            "id": "FactorB",
            "label": "Factor B (Syn)",
            "category": "CategoryA"
        },
        ...
    ],
    "results": {...},
    "provided": [
        {
            "source": {...},
            "questionnaire": {
                "ProvidedSearchQuestion": {
                    "answer": [
                        "Factor C",
                        "Factor D (Syn)"
                    ],
                    "answerDescriptor": "Label"
                }
            }
        }
    ],
    "conversationId": "eca7e457-9f08-4612-8b6c-04a6365e744f",
    "completed": false
}
