Recommended systems

ReFlex supports a variety of external systems. An overview of the recommended systems is listed below. Those systems are tested by the ReFlex team. It is possible that ReFlex is also working on different systems but those are not tested and therefore cannot be supported.

Operating systems

OS family OS version
Linux Ubuntu LTS 20.04
Linux Debian 10.4
Linux RedHat 8.2
Windows Server 2019

Java

runtime JDK version
Oracle 11 (latest/LTS; today: 11.0.8)
AWS Corretto 11 (latest/LTS; today: 11.0.8.10.1)

Note that for best performance the RAS can be configured to use a Graal compiler at runtime. This is only available with the Open JDK / Corretto JDK.

Application server

type version
Apache Tomcat 9.0 (latest), today 9.0.37

Database

type version component
Oracle 19c CEP
MS SQL Server 2019 CEP, UWB
MySQL 8.0.(latest) CEP
PostgreSQL 12.(latest) UWB

MS SQL Server is only supported for the Windows Server environment

User Management / LDAP / AD

type version component
Keycloak 11.0.2 UWB, RKE, ATT

Browser

The ReFlex team tests and supports the two most recent browser versions available at release time.

type version device ReFlex component
Chrome latest PC all UI related components
Firefox latest PC all UI related components
MS Edge (Chromium-based) latest PC all UI related components
Safari latest iPhone 6 and higher, IOS 13 CEP
Safari latest iPad 3 Retina and higher, IOS 13 CEP, MI Dashboard
Chrome latest Tablet Android 10 CEP, MI Dashboard
Chrome latest Smartphone Android 10 CEP, MI Dashboard

Note for Chrome/Firefox/MS IE & Edge: Recommended screen resolution should be at least 1376 × 768 pixels.
