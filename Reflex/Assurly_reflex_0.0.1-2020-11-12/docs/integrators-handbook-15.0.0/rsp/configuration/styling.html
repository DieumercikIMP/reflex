Styling RSP

RSP styling files that are mentioned in this chapter:

$REFLEX_CONFIG_ROOTDIR/rsp/cedentConfig/images/${cedentId}-logo-left.png
$REFLEX_CONFIG_ROOTDIR/rsp/cedentConfig/styles/${cedentId}.css

For the different partnerIDs, that are important for the numbering scheme of the files, please refer to the chapter about customization.

Coloring Scheme

The CSS file for general styling changes can be found in $REFLEX_CONFIG_ROOTDIR/rsp/cedentConfig/styles.

In the existing CSS file (00000.css) you will find comments above each property, how it is used and where. The basic layout should not be changed or manipulated from here, only the coloring scheme and font characteristics.
The existing CSS file can either be edited/exchanged, or you can add your own CSS file for another partnerID. For further information about partnerIDs refer to the customization chapter of this document.

Logo

The logo displayed in the top left corner of the RSP can be found in $REFLEX_CONFIG_ROOTDIR/rsp/cedentConfig/images. The existing image can either be exchanged by an one of your insurance companies logo, or you can add your image under another partnerID.
For further information about partnerIDs refer to the customization chapter of this document.
