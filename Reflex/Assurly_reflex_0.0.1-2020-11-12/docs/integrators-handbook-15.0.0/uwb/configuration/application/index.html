Application configuration

This section documents how the UWB instance needs to be configured prior to system start.

  • Application - basic technical configuration of an UWB instance (e.g. database connectivity, …)
  • Mail Service Configuration - additional technical configuration for connecting to a mail server
  • Permission Configuration - setting up UWB permissions in external authorization server (e.g. Keycloak)
  • Logging - configuring the UWB behaviour for logging messages (verbosity, target)

After the initial setup of the application, at least one office needs to be configured. This is described in chapter Office configuration.
