Cover types

A cover type represents a specific risk cover that is sold by the insurance company. It is linked to exactly one authority type like Life or Disability. At least one cover type is required for the UWB to function properly.

Cover types can be created with the following statement:

insert into COVER_TYPES (ID, OFFICE_ID, NAME, AUTHORITY_TYPE, ASCENT_PRODUCT, POSITION, DEACTIVATED, VERSION) values
  (next value for covertype__seq, <office-id>, 'Level term assurance', 'LIFE', 'LIFE', 10, 0, 0),
  (next value for covertype__seq, <office-id>, 'TPD any', 'DISABILITY', 'TPD_ANY', 50, 0, 0);
Field Description Type Required Mutability
ID Unique technical identifier. Use sequence for id generation. See SQL snippet. bigint Once
Office ID Specifies the relation to an existing office. A cover type always belongs to exactly one office. bigint Once
Name The default name of the cover type. A cover type has a (default) name and optionally a local and/or display name. The local name is used for document creation, e.g. decision letter. The display name is used for the UI. The default name is a fallback for the local and the display name. All names can be changed via the admin portal. nvarchar(120) Runtime
Authority Type Type of risk associated with the cover type. See below for possible values. varchar(40) Runtime
Ascent Product Ascent product associated with the cover type. See below for possible values. varchar(40) Runtime
Position Position in ascending order in which the cover types are displayed in the UI. int Runtime
Deactivated Should always be false when creating a new cover type. bit Runtime
Version Technical field used for optimistic locking. Always 0. int Once

The following authority types are supported:

Authority Type Description
CDI Immediate Care Annuities
CRITICAL_ILLNESS Critical Illness Cover / Dread Disease
DISABILITY Total Permanent or Temporary Disability, Waiver of Premium (Disability)
ENHANCED_ANNUITIES Enhanced / Impaired Annuities
HEALTH Inpatient or Outpatient health insurance, Dental, Hospital Cash
LIFE Term Life, Endowment, Whole Life
LONG_TERM_CARE Long Term Care
PERSONAL_ACCIDENT Accidental Death Benefit, Accident

The following list shows all existing Ascent products, however, the actually supported Ascent products depend on the selected region:

Authority Type Description
LIFE Term Life, Endowment, Whole Life
ADB Accidental Death Benefit
CI Critical Illness / Serious Illness / Dread Disease
TPD_OWN Total Permanent Disability own occupation
TPD_ANY Total Permanent Disability any occupation
WP Waiver of Premium
D4 Income Protection / PHI (4 weeks waiting period)
D13 Income Protection / PHI (13 weeks waiting period)
D_13_D26 Income Protection / PHI (26 weeks waiting period)
D4_UP Income Protection / PHI
HOSPITAL Hospital Cash
LTC Long Term Care
LTC_W60 Long Term Care (60 days waiting period)
